# 🤖 AI Assistant POC – RAG &  Agent Autonome

## 🎯 Objectif

Développement d’un POC d’assistant intelligent basé sur des architectures modernes en IA (RAG + agents), capable de répondre à des requêtes complexes et d’assister la prise de décision.

---

## ⚙️ Architecture du projet

Le projet repose sur deux briques principales :

### 🔹 1. Système RAG (Retrieval-Augmented Generation)

* Génération d’embeddings avec SentenceTransformers (MiniLM)
* Stockage vectoriel avec ChromaDB
* Orchestration avec LangChain
* Indexation de documents spécialisés

### 🔹 2. Agent IA autonome

* Développement avec smolagents
* Traitement de requêtes complexes
* Intégration d’un système RAG
* Combinaison dynamique de plusieurs sources
